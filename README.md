# customer_behavior_analysis
🔍 Overview

This project focuses on analyzing customer behavior using a combination of Python, SQL, and Power BI. The workflow includes data loading, exploratory data analysis (EDA), data cleaning, querying using MySQL, and building an interactive dashboard to derive business insights.

📁 Dataset

The dataset contains customer-related information such as:

Demographics (age, gender, location)
Purchase details (amount, category, frequency)
Customer behavior (reviews, ratings, subscription status)

🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn) – Data analysis & EDA
MySQL Server – Data storage & querying
Power BI – Dashboard creation & visualization
Jupyter Notebook / VS Code – Development environment

⚙️ Project Steps
1. Data Loading (Python)
 Imported dataset using Pandas
 Checked structure, data types, and missing values
2. Exploratory Data Analysis (EDA)
 Analyzed distributions, correlations, and trends
 Visualized key metrics using plots and charts
3. Data Cleaning
 Handled missing values
 Removed duplicates
 Standardized column formats
4. SQL Analysis (MySQL)
 Loaded cleaned data into MySQL
 Wrote queries to extract insights such as:
 Total customers
 Revenue by category
 Purchase frequency patterns
5. Dashboard Creation (Power BI)
 Connected Power BI to dataset
 Built interactive dashboard with:
 KPI cards (customers, average purchase, rating)
 Category-wise revenue and sales
 Filters (gender, category, subscription)

📊 Dashboard

The Power BI dashboard provides:

A clear overview of customer metrics
Interactive filters for deeper analysis
Visual insights into sales and customer behavior

📈 Results & Insights
Identified top-performing product categories
Analyzed customer purchasing patterns
Observed differences in behavior based on demographics
Derived actionable insights for business decisions
